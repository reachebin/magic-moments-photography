# Magic Moments Photography (GitHub Pages)

## Run locally
Just open `index.html` in a browser, or use VS Code Live Server.

## Publish on GitHub Pages
1. Create a new GitHub repo (e.g., `magic-moments-photography`)
2. Upload these files: `index.html`, `styles.css`, `script.js`, `README.md`
3. Go to **Settings â†’ Pages**
4. Under **Build and deployment**, set:
   - Source: **Deploy from a branch**
   - Branch: **main** / **root**
5. Save. Your site will be available at:
   `https://YOUR-USERNAME.github.io/magic-moments-photography/`

## Replace content
- Update hero image in `styles.css` under `.hero__media`
- Replace portfolio images in `index.html`
- Edit testimonials in `script.js`

## Contact form
GitHub Pages is static, so forms need a service.
- Formspree: replace `https://formspree.io/f/yourFormId` in the form `action`.
